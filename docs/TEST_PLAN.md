1. Запусти проект локально: `npm run dev`

3. Ставь статус: ✅ ОК | ❌ БАГ | ⚠️ Мелочь
4. Если баг — опиши что видишь (скриншот приветствуется)
5. После прохождения — отдай этот файл мне, я починю всё что нужно
