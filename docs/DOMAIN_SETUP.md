# Привязка домена opora-migration.ru

## Текущая конфигурация

- **Регистратор домена:** Timeweb (аккаунт ck68396)
- **Хостинг сайта:** Beget Cloud VPS (193.42.124.57)
- **DNS:** управляется через панель Timeweb

Домен и хостинг — разные вещи. Домен зарегистрирован на Timeweb, но сайт работает на сервере Beget. Связь между ними — через DNS A-запись.

## Как это работает

```
Пользователь набирает opora-migration.ru
    → DNS (Timeweb) отвечает: IP = 193.42.124.57
    → Браузер обращается к 193.42.124.57 (Beget VPS)
    → Nginx → Next.js → сайт
```

## Текущие DNS-записи

| Тип | Хост | Значение | Назначение |
|-----|------|----------|------------|
| A | opora-migration.ru | 193.42.124.57 | Сайт → Beget VPS |
| MX | opora-migration.ru | mx1.timeweb.ru | Почта (не трогать!) |
| MX | opora-migration.ru | mx2.timeweb.ru | Почта (не трогать!) |
| TXT | opora-migration.ru | v=spf1... | Почта (не трогать!) |

## Если нужно изменить IP сервера

1. Войти в https://hosting.timeweb.ru (аккаунт ck68396)
2. Домены и SSL → opora-migration.ru → Редактор DNS
3. Найти A-запись → нажать три точки → Редактировать
4. Заменить IP на новый
5. Сохранить. Обновление займёт 5-30 минут.

**Важно:** MX и TXT записи отвечают за почту — их не трогать!

## SSL-сертификат

SSL (HTTPS) настроен через Let's Encrypt на сервере Beget. Сертификат продлевается автоматически через certbot.

Если нужно перевыпустить вручную:
```bash
ssh root@193.42.124.57
certbot renew --force-renewal
```
